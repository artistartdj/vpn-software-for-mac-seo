## FAQ

**1) Does this support a Kill Switch on macOS?**  
Yes. Enable **Kill Switch** in settings to block traffic if the VPN disconnects.

**2) Is it actually no-logs?**  
The intent is a strict **No-Logs policy**—no tracking of your browsing activity.

**3) Will this slow my connection down?**  
Any VPN adds overhead. The goal here is **high speed** and a **stable connection** with sensible defaults.

**4) How do I switch regions/servers?**  
Open the server list, pick a region from **Global Servers**, then reconnect.
